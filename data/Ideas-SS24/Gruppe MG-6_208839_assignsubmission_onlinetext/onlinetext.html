Als Projektidee wollen wir eine File-Sharing-Website, welche auf das Teilen von Schachanalysen (hauptsächlich im .pgn-Format) fokussiert ist, kreieren. Der Hauptnutzen dieser Website ist das Teilen von Eröffnungsanalysen, kommentierten Partien, thematischen Studien u.Ä.

Die Nutzer legen ein Profil/Konto an und können subsequent Dateien hochladen bzw. die Dateien anderer Nutzer herunterladen. Die heruntergeladenen Dateien können bewertet und kommentiert werden. Nutzer können auch Änderungen zu einer Datei vorschlagen, welche (nach einer Überprüfung?) zu einem Versionsverlauf hinzugefügt werden. Auf der Website ist ein pgn-Viewer integriert, welcher eine Vorschau der Dateien anbietet. Zusätzlich gibt es eine Suchfunktion, mit der man nach verschiedensten Kriterien filtern kann (ECO-Code, Thema, etc). Nutzer können außerdem eine private Bibliothek anlegen, um ihre Dateien online zu speichern.

Jede Datei wird vor dem Hochladen automatisch auf Malware überprüft. Es wird ein Reporting-System implementiert, um z.B. Copyright-Claims effektiv bearbeiten zu können.
